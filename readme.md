# Task Manager

Task Manager is a Java-based CLI application for managing tasks. It uses MySQL for persistent storage and runs in Docker containers.

## Features
- Add tasks
- List tasks
- Complete tasks
- Delete tasks

## Prerequisites
- Docker
- Docker Compose
- Java 17
- Maven
